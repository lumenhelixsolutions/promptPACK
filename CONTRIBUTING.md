# Contributing

Keep changes focused and testable.

## Local setup

```bash
npm install
npm run ci
```

## Rules

1. Preserve local-first behavior.
2. Do not add network calls without discussion.
3. Do not add automatic insertion, sending, publishing, or scraping.
4. Add or update preset tests when objective logic changes.
5. Route high-risk or irreversible actions to the human.

## Pull requests

Use the PR template and include preset gate results.
