# promptPACK v0.1.1 UX Trust Pass

This release improves trust and control without bloating the extension.

## Added

- Editable locked phrases
- Editable must-preserve facts
- Target platform selector
- Aggression level selector
- Result report panel
- Copy report
- Copy output + report
- Preservation evidence pills
- Stricter preset preservation checks

## Why

The lab established that prompt optimization should behave like a compiler/compressor/triage system. The extension now exposes the minimum controls needed for users to state what must survive transformation.

## Still local-first

No network calls were added. Selected-text import, copy, and insertion remain user-triggered.
