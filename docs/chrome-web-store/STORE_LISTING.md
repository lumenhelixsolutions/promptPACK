# Chrome Web Store Listing — promptPACK

## Extension name

promptPACK

## Short description

Local-first prompt compression with preservation controls and human-in-the-loop routing.

## Detailed description

promptPACK is a local-first Chrome side-panel extension for safer LLM prompt optimization.

It helps you decide whether a prompt should be compressed, left alone, or converted into a structured handoff packet for coding, research, audit-sensitive work, or local-model workflows.

promptPACK is not a generic summarizer. It is designed to preserve what matters.

## Key features

- Objective-aware prompt compression
- No-op detection when rewriting is not useful or safe
- Editable locked phrases
- Editable must-preserve facts
- Target platform selector
- Aggression level selector
- Human-in-the-loop decision routing
- Built-in preset test gate
- Copy output
- Copy output + result report
- User-confirmed insertion into active fields

## Privacy-first behavior

- No network calls
- No remote code
- No account required
- No analytics
- No automatic scraping
- No automatic insertion
- Text is processed locally in the browser extension runtime

## Current status

Developer preview / prototype.

promptPACK is intended for users who want more control over prompt compression and prompt handoff workflows. Always review generated prompts before sending them to an LLM.
