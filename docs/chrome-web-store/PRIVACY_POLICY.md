# Privacy Policy for promptPACK

Effective date: 2026-05-19

promptPACK is a local-first Chrome extension.

## Data collection

promptPACK does not collect, sell, transmit, or share personal data.

## Network activity

promptPACK does not make network calls.

## Text processing

Text entered into promptPACK, selected from a webpage, or inserted into an editable field is processed locally in the browser extension runtime.

## Selected text

promptPACK only reads selected text after the user clicks the "Use selected text" button.

## Insertion

promptPACK only inserts text after the user clicks "Insert into active field." It does not auto-send, auto-publish, or auto-submit anything.

## Storage

The current prototype does not require cloud storage or accounts. If future versions add optional local history or sync features, this policy should be updated before release.

## Permissions

promptPACK uses Chrome extension permissions only to provide its side-panel workflow, read user-selected text after user action, and insert output into the active editable field after user action.

## Contact

For privacy questions, open an issue in the project repository.
