# promptPACK Brand Guide

## Official name

promptPACK

Use lowercase `prompt` and uppercase `PACK`.

## Product family

| Name | Meaning |
|---|---|
| promptPACK | Main Chrome extension |
| promptPACK Lab | Methodology/test harness |
| promptPACK Gate | Preset tests |
| promptPACK Routes | Human-in-the-loop decision routing |
| promptPACK Guard | Preservation and risk layer |

## Tagline

Compress the prompt. Preserve the point.

## Short description

Local-first Chrome extension for objective-aware prompt compression, no-op detection, must-fact preservation, and human-in-the-loop prompt handoff.

## Voice

Clear, technical, and safety-aware. Avoid hype that makes it sound like magic. The product’s credibility comes from restraint: it knows when not to rewrite.
