# Viral Repo Checklist

A viral repository is not just useful. It is instantly legible, easy to try, and easy to share.

## Above the fold

- [x] Clear name
- [x] One-line value proposition
- [x] Visual identity
- [x] Badges
- [x] Quick install
- [x] Expected result

## Trust

- [x] Local-first privacy statement
- [x] No network calls
- [x] Preset gate
- [x] CI workflow
- [x] Security policy
- [x] Contributing guide

## Shareability

- [x] Social preview asset
- [x] Suggested topics
- [ ] Demo GIF
- [ ] Screenshots
- [ ] Launch post
- [ ] Short explainer video

## Developer adoption

- [x] `npm run ci`
- [x] Simple file structure
- [x] Roadmap
- [x] Issue templates
- [x] PR template
