# promptPACK Product Brief

## One-liner

promptPACK is a local-first Chrome extension that compresses prompts when safe, no-ops when safer, and turns coding/research/local-model tasks into reviewable handoff packets.

## Hook

Most prompt tools rewrite everything. promptPACK knows when not to.

## User problem

LLM prompts are getting longer, messier, and more expensive. Users need help shortening prompts without losing the details that matter.

## Differentiator

promptPACK combines objective detection, editable preservation controls, no-op as a valid result, human-in-the-loop routing, local-first browser behavior, and preset gate testing.

## Commercial path

Open-source core, commercial polish later: Chrome Web Store distribution, optional pro presets, team prompt policies, local history/export packs, model-specific optimization recipes, and enterprise policy templates.
