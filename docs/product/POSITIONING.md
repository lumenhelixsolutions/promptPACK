# Positioning

## Category

Prompt safety and compression utility.

## Tagline

Compress the prompt. Preserve the point.

## Core message

promptPACK is not a generic summarizer. It is a prompt compiler-style assistant that decides whether to compress, no-op, or create a handoff packet.

## What it refuses to be

- not an auto-scraper
- not an auto-sender
- not a remote prompt service
- not a black-box rewrite tool
- not a replacement for human review on high-stakes tasks
