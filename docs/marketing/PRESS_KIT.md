# promptPACK Press Kit

## Name

promptPACK

## Tagline

Compress the prompt. Preserve the point.

## Short description

Local-first Chrome extension for objective-aware prompt compression, no-op detection, must-fact preservation, and human-in-the-loop prompt handoff.

## Launch hook

Most prompt tools rewrite everything. promptPACK knows when not to.

## Key claims

- Local-first prototype
- No network calls
- No auto-insert
- Built-in preset gate passing 6/6
- Editable preservation controls
- Human-in-the-loop routing
